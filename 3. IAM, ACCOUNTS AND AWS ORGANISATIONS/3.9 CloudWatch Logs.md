- **CloudWatch Logs** is a service which can accept logging data, store it and monitor it.

![alt text](<Screenshot 2025-08-17 161140.png>)

- **Logging data** is a piece of information data, and a timestamp.

![alt text](<Screenshot 2025-08-17 161816.png>)

- **Log events** are stored inside log streams and log streams are essentially a sequence of log events from the same source.

- **Log groups** are containers for multiple log streams for the same type of logging.

Each log stream can represent one source.

Log groups are also a place that stores configuration settings.
