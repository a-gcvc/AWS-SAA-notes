- When application interacts with S3 to retrieve any data, the entire object or stream of objects are delivered to the application.

Application receives everything. It can either accept it as a whole or it can perform its own filtering.

Any filtering performed by the application is performed inside the application.
It doesn't impact the cost or performance.
