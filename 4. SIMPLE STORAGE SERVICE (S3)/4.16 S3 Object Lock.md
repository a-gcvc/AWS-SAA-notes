- WORM: you can set it so that object versions once created can't be overwritten or deleted

![alt text](<Screenshot 2025-08-20 192239.png>)

- **Governance** mode: you still set a retention period, and while active the object version cannot be deleted ot changed in any way, but you can grant special permissions to allow these to be changed.

Useful for:
    - if you want to prevent accidental deletion
    - if you have process reasons or governance reasons to keep object versions 
    - you might use it as a test of settings before picking the compliance mode
